First public beta as **Matter Cameras Bridge** — Matter-compatible RTSP/ONVIF bridge for Matter 1.5 cameras on SmartThings and other hubs.

### Highlights

- Web UI with Matter pairing, camera providers (UniFi Protect, Reolink, ONVIF, Tapo/Sonoff, manual RTSP)
- Live view WebRTC, snapshots, motion routines (Zone Management + OccupancySensing)
- Optional GitHub release notifications and `scripts/self-update.sh` for updates
- Mechanical PTZ (beta), person presence sensor, Reolink spotlight endpoint

### Install

```bash
git clone https://github.com/patricktd/MatterCameras.git
cd MatterCameras
bash scripts/setup.sh
```

See [docs/INSTALL.md](https://github.com/patricktd/MatterCameras/blob/main/docs/INSTALL.md).
